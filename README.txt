Aquesta pràctica s'ha desenvolupat amb el llenguatge de programació Python 3.9.0.
Per tant, per poder executar-la, s'ha de tenir instal·lat a l'ordinador. Per
fer-ho, es tan simple com dirigir-se a la plana web oficial de Python, triant
aquesta darrera versió que va sortir el passat 5 d'octubre del 2020. 
Els links de descarrega (pels diversos sistemes operatius) es poden trobar a la
següent url: 

	https://www.python.org/downloads/release/python-390/

Una vegada instal·lada aquesta distribució de Python, el programa ja pot executar-se.
Es pot fer molt simple desde la consola (tenguen en compte que hem de tenir el python
al conjunt de les variables d'entorn). Per fer-ho simplement hem de introduir
la següent comanda:

	C:\Practica1\CodiFont> py Principal.py
